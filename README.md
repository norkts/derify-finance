# derify-finance

Contains the source code for the [derify.finance](http://derify.finance) homepage.

## Development

### setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```

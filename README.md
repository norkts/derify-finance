# derify-finance

This is the repo for source code of the [derify.finance](https://derify.finance) homepage.

## Development

### Dependencies setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
